Help/Tips

★ Philly's BigFiles Manipulator is quite easy to understand and use
★ Find full documentation in the "About & Help" section of the application


[Features]

- Search through data files like .txt, .csv files using a keyword or multiple keywords
- Can search through JSON files and give readable and compact output
- Extract exixsting eMail-addresses in files if there are any
- MXR Sorting for eMail-Addresses
- Split Large Files by size or by number of lines
etc.


[Requirements]

★ Python 3.x or later
★ At least 2 Gigabyte of RAM
★ Disk Space of 15MB


[How To Install]

1. Download Philly's BigFiles Manipulator (either the free or the full version). It should download as a .zip file
2. If you don't already have python 3.x installed on your machine,
please visit https://www.python.org/downloads/ and download the latest version of python 3 for your operating system.
3. Extract the contents of the "PBFM.zip" file and double click on "install.bat"
4. After successfull installation of Step 3, double click on "run.bat" if you use a Windows machine
5. For Linux & Mac users simply go to your terminal and run the command, "python3 bigfiles.py"
