[Features]

-Manipulate records in your data files like extract data (eMail Addresses, records etc.) with or without keywords in seconds
-Parse through BigFiles of up to 50GB
-Sort e-Mail Addresses from a file/files into their various MXRecords. (e.g Rackspace, Gmail etc)
-Split Large files into smaller files either by size (e.g splitting a 2GB file into five[5] files of 400mb each) or by the number of lines each smaller file should contain
-Work with multiple files at the same time © (Available in paid version)
- Search through data files like .txt, .csv files using a keyword or multiple keywords
- Parse JSON files and give readable and understandable output (Available in paid version)
- Extract exixsting eMail-addresses in selected files if there are any...
etc.


[Requirements]

★ Python 3.x or later
★ At least 2 Gigabyte of RAM
★ Disk Space of about 12MB for the software itself


[How To Install]

1. Download Philly's BigFiles Manipulator (either the free or the full version). It should download as a .zip file
2. If you don't already have python 3.x installed on your machine,
please visit https://www.python.org/downloads/ and download the latest version of python3 for your operating system.
3. Extract the contents of the "PBFM().zip"
4. Open the "Installation" folder
_______a) double click the installation file for your system eg. windows_run.bat for WindowsOS
(Note: Latest versions of windows may prevent .bat files from running, so you may have to disable the UAC so they can run)
OR
_______b) open your terminal(command prompt on windows), navigate to the software's main folder ('cd /path/to/folder') and run: "pip install -r requirements.txt" (make sure you have pip installed)

If you missed any installation, it will tell you what you have to install

5. Once successfully installed, for the last step, you can either:
_______a) open the 'run' folder and double click on the file that suits your platform eg. 'mac_start.command' for MacOS
OR
_______b) open your terminal(command prompt on windows), navigate to the software's main folder ('cd /path/to/folder') and run: "python bigfiles.py" or "python3 bigfiles.py" command depending on how python3 is installed on your machine
